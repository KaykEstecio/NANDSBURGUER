# VS Code Settings for NANDS Burguer Project

```json
{
  // Editor
  "editor.defaultFormatter": "esbenp.prettier-vscode",
  "editor.formatOnSave": true,
  "editor.codeActionsOnSave": {
    "source.fixAll.eslint": true
  },
  "editor.tabSize": 2,
  "editor.insertSpaces": true,
  "editor.trimAutoWhitespace": true,
  "files.trimTrailingWhitespace": true,
  "files.insertFinalNewline": true,

  // TypeScript
  "typescript.tsdk": "node_modules/typescript/lib",
  "typescript.enablePromptUseWorkspaceTsdk": true,

  // Recommended Extensions
  "extensions.recommendations": [
    "esbenp.prettier-vscode",
    "dbaeumer.vscode-eslint",
    "ms-vscode.vscode-typescript-next",
    "Prisma.prisma",
    "bradlc.vscode-tailwindcss"
  ]
}
```

## Extensions Recomendadas

1. **Prettier** - Code Formatter
2. **ESLint** - JavaScript linter
3. **Prisma** - ORM support
4. **Tailwind CSS IntelliSense** - CSS utility classes
5. **Thunder Client** - API testing
6. **REST Client** - HTTP testing

## Tips

- Use `Cmd+Shift+P` (Mac) ou `Ctrl+Shift+P` (Windows) para command palette
- Prettier formata ao salvar automaticamente
- ESLint mostra erros em tempo real
- TypeScript mostra erros de tipo
